# Text Analyser in C++

## 📖 Description
This is a simple Text Analyser program written in C++. 
It performs basic text operations such as:
- Counting vowels
- Counting consonants
- Counting words
- Counting digits
The project is built using basic C++ concepts like functions, loops, conditionals, and switch-case.

## 🚀 Features
- Count digits in a sentence if exists
- Count vowels and consonants
- Count words
- Menu-driven program

## 🛠 Technologies Used
- C++
- Standard Template Library (STL)
- iostream
- string

- ## ▶ How to Run

1. Clone the repository
   git clone https://github.com/jotirampatil1301/text-analyser.git

2. Open in any C++ compiler (VS Code / CodeBlocks / g++)

3. Compile
   g++ main.cpp -o textanalyser

4. Run
   ./textanalyser
